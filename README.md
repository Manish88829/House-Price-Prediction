# 🏠 House Price Prediction

A Machine Learning based House Price Prediction web application built using Python, Flask, Scikit-Learn, HTML, CSS and JavaScript.

## Features

- Predict house prices using Machine Learning
- Modern responsive UI
- About Page
- Contact Page
- FAQ Page
- Price Trends Page
- Beautiful Result Page

## Tech Stack

- Python
- Flask
- Scikit-Learn
- Pandas
- NumPy
- HTML5
- CSS3
- JavaScript

## Project Structure

```
House-Price-Prediction
│
├── app.py
├── train_model.py
├── requirements.txt
├── model/
├── data/
├── static/
├── templates/
```

## Installation

```bash
git clone https://github.com/Manish88829/House-Price-Prediction.git

cd House-Price-Prediction

pip install -r requirements.txt

python app.py
```
